ℹ️ About Task Organizer

Welcome to the Fun Task Organizer! This is a simple, interactive, and colorful app to help you manage your daily tasks in a fun way.

Features:

  • ✅ Add & Remove Tasks
  • 🎉 Mark Tasks as Completed
  • 💾 Saves Tasks Even After Refresh
  • ✨ Smooth Animations & Colorful UI
